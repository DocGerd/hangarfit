# Tow-planner staging apron (#412) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task (inline). The change is a tightly-coupled cluster around `towplanner.py` + `loader.py` + `models.py`; subagent-per-task would thrash the shared files and lose the determinism context, so execute inline. Steps use checkbox (`- [ ]`) syntax.

**Goal:** Model a bounded fixed-depth staging apron (`y ∈ [−apron_depth_m, 0)`) in front of the door so each plane's tow path starts *outside* the hangar and slides in through the door, while keeping the ADR-0003 byte-identical-plan contract intact (`apron_depth_m = 0`/absent ⇒ today's plan, byte-for-byte).

**Architecture:** `apron_depth_m` is a new optional `Hangar` scalar (default `0`). It rides on the `Hangar` object, so it reaches all three planner change-sites for free (`plan_fill` reads `target.hangar`; `entry_poses(slot, hangar)` and `_mover_motion_bounds_conflict(…, hangar)` already receive it; `solver.solve` builds candidates from `scenario.hangar`). Every changed function keeps its **original code path verbatim behind an `apron_depth_m > 0` gate**, so depth-0 output is structurally byte-identical. The CLI `--apron-depth N|auto` threads through `load_scenario(apron_depth=…)` exactly like `--max-carts` — never as a `solve()` kwarg. The opt-in `auto` value derives `≈ max(plane fore-aft length) + max(turn_radius_m)` from the fleet at load time.

**Scope decisions (user-locked 2026-06-07):** include `auto` now; **add reverse-entry seed headings now** (≈180°, gated on depth>0 — enables #263 nose-out routing); **defer** the render-only viewer apron-ground field to a follow-up (the slide-in *motion* already works with zero `scene/v1` change). `collisions.check` is **untouched**.

**Tech Stack:** Python 3.12, shapely (existing part polygons), pytest, argparse. No new deps.

```python
def _plane_fore_aft_length_m(aircraft: Aircraft) -> float:
    """Fore-aft (local-x) extent of a plane from its parts.

    Approximates per-part rotation away (struts) — adequate for the opt-in
    ``auto`` apron depth (a convenience knob, not a tight bound). RNG-free.
    """
    fronts = [p.offset_x_m + p.length_m / 2.0 for p in aircraft.parts]
    backs = [p.offset_x_m - p.length_m / 2.0 for p in aircraft.parts]
    return max(fronts) - min(backs)


def derive_apron_depth(fleet: Mapping[str, Aircraft]) -> float:
    """Fleet-derived staging-apron depth — the opt-in ``auto`` value (ADR-0021).

    ``≈ max(plane fore-aft length) + max(own-gear turn radius)``: the run-up room
    one plane needs to align and slide in. A pure function of the fleet (RNG-free),
    so the resolved depth is deterministic. Empty fleet ⇒ ``0.0``.
    """
    if not fleet:
        return 0.0
    max_len = max(_plane_fore_aft_length_m(a) for a in fleet.values())
    radii = [a.turn_radius_m for a in fleet.values() if a.turn_radius_m is not None]
    return max_len + (max(radii) if radii else 0.0)
```

**Rule (per-footprint straddle gate):** a footprint that has vertices both at `y<0` and `y>0` *crosses* the front wall, so its `y<0` vertices must pass through the door opening (#411 verbatim). A footprint **wholly** in front of the wall (all `y≤0` — staged on the apron) does not cross, so the door-gate does not apply — only the apron south bound `y ≥ −apron_depth_m` and the unchanged side/back walls. At `apron_depth_m == 0` the apron branch is unreachable and the rule is the verbatim #411 per-vertex door-gate.

def _resolve_apron_depth(
    value: Any, fleet: Mapping[str, Aircraft] | None, *, field_name: str = "apron_depth_m"

def _apron_depth_arg(value: str) -> float | str:
    if value.strip().lower() == "auto":
        return "auto"
    try:
        f = float(value)
    except ValueError:
        raise argparse.ArgumentTypeError(
            f"--apron-depth must be a number or 'auto', got {value!r}"
        )
    if not math.isfinite(f) or f < 0:
        raise argparse.ArgumentTypeError(
            f"--apron-depth must be a finite, non-negative number or 'auto', got {value!r}"
        )
    return f
